## Local Development

### 🧑‍🤝‍🧑 Clone the project

```sh
https://github.com/joysofcode/sveltekit-web-scraping.git
```

### 📦️ Install dependencies

```sh
pnpm i
```

### 💿️ Run the development server

```sh
pnpm run dev
```
